初級から上級まで、Claude Code を活用して構築したスキルと自動化の記録。

初級

文字数カウント
スキルの活用

/char-count スキルを使い、スペース・改行の有無で4パターンの文字数を自動集計。

/char-count Skill実行 Python

中級

セッション管理と
自己紹介ファイル作成

Claude Code のセッションを整理し、README.md として自己紹介ファイルを自動生成。

セッション管理 README 生成

中級

カスタムスキル「mojisuu」の作成・リネーム

char-count スキルをゼロから作成し、mojisuu にリネーム。フォルダ名と SKILL.md を同時更新してスキルリストに即反映。

カスタムスキル mojisuu SKILL.md

上級

ブログ構成生成スキル
/blog-designer

テンプレートを読み込み、指定テーマでブログ記事を自動生成。文字数カウントまで一気通貫で処理。

/blog-designer テンプレート活用 文字数検証

上級

セキュリティレビュー
スキル /code-reviewer

SubAgent を用いた自動コードレビュースキルを開発。SQL インジェクション・MD5 脆弱性を検出し、7件の修正を自動適用。

/code-reviewer SubAgent セキュリティ 自動修正
