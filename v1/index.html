初級

文字数カウントスキルの活用

/char-count スキルを使い、複数パターンの文字数を自動集計。スペース・改行の有無で4種類の結果を出力。

中級

セッション管理と自己紹介ファイル作成

Claude Code のセッションを整理し、README.md として自己紹介ファイルを自動生成。

セッション管理 README 生成
中級

カスタムスキル「mojisuu」の作成・リネーム

char-count スキルをゼロから作成し、スキル名を mojisuu に変更。フォルダ名と SKILL.md の name フィールドを同時更新し、スキルリストへ即座に反映させた。

カスタムスキル mojisuu SKILL.md
上級

ブログ構成生成スキル /blog-designer

テンプレートを読み込み、指定テーマでブログ記事を自動生成。文字数カウントまで一気通貫で処理するスキルを開発。

/blog-designer テンプレート活用 文字数検証
上級

セキュリティレビュースキル /code-reviewer

SubAgent を用いた自動コードレビュースキルを開発。SQL インジェクションや MD5 ハッシュ等の脆弱性を検出し、7件の修正を自動適用。

/code-reviewer SubAgent セキュリティ 自動修正
