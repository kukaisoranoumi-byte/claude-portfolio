スキルの活用

セッション管理 README 生成 Markdown

▲▲▲ 上級

ブログ構成生成スキル
/blog-designer

テンプレートを読み込み、指定テーマでブログ記事を自動生成。 文字数カウントまで一気通貫で処理するフルスタックなスキルを開発。

/blog-designer テンプレート活用 文字数検証
▲▲▲ 上級

セキュリティレビュー
スキル /code-reviewer

SubAgent を用いた自動コードレビュースキルを開発。 SQL インジェクション・MD5 脆弱性を検出し、7件の修正を自動適用。 マルチエージェント構成でレビューの精度と速度を両立。

/code-reviewer SubAgent セキュリティ 自動修正

Journey

学習の軌跡

初級から上級へ。段階的にスキルを積み上げてきた道のり。

初級
既存スキルの活用
/char-count スキルを使いこなし、4パターンの文字数を自動集計。 Claude Code の基本操作とスキル実行の流れを習得。
中級
セッション管理とカスタムスキル作成
セッション履歴の整理・README 自動生成を経て、 char-count スキルをゼロから作成し mojisuu にリネーム。 SKILL.md の構造設計とスキルリスト管理を実践。
上級
SubAgent によるフルオートメーション
/blog-designer でコンテンツ生成を自動化し、 /code-reviewer では SubAgent を用いて SQL インジェクション・ MD5 脆弱性を検出・修正するセキュリティパイプラインを構築。
